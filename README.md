# Solana Workflow

> Create automated workflows for the Solana ecosystem using natural language and AI.

Solana Workflow is a platform that allows users to create, manage, and execute automated workflows on Solana without writing code. Users describe what they want to automate in plain language, and the platform transforms those instructions into structured workflows that can be monitored and executed automatically.

## Overview

Solana Workflow combines artificial intelligence and blockchain automation to make Web3 more accessible.

Instead of writing scripts or configuring complex tools, users can simply describe what they want:

> "Notify me when SOL reaches $200."

The platform interprets the request, generates a workflow, monitors the required conditions, and executes the corresponding action when the trigger is activated.
